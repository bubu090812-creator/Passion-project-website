# My Passion Project Website

A modern, responsive website template for your passion project.

## Features

✨ **Modern Design** - Clean and professional layout with smooth animations
📱 **Responsive** - Works perfectly on desktop, tablet, and mobile devices
🎨 **Customizable** - Easy to personalize with your own content and colors
⚡ **Performance** - Lightweight and fast-loading pages
🔄 **Smooth Interactions** - Animated elements and smooth scroll navigation

## Structure

```
├── index.html          # Main HTML file
├── style.css           # Styling and responsive design
├── script.js           # Interactive features and animations
└── README.md          # This file
```

## Sections

- **Navigation** - Sticky navigation bar with smooth scrolling
- **Hero Section** - Eye-catching introduction with call-to-action
- **About Section** - Tell your story and highlight key features
- **Projects Section** - Showcase your work with project cards
- **Contact Section** - Contact form for visitors to get in touch
- **Footer** - Links and social media connections

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Customize the content to match your passion project
4. Update the colors, images, and text as needed

## Customization Guide

### Update Navigation
Edit the project name in `index.html`:
```html
<div class="nav-brand">Your Project Name</div>
```

### Change Colors
Modify the gradient colors in `style.css`:
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

### Add Your Projects
Edit the project cards in the Projects section with your own details.

### Update Social Links
Change the social media URLs in the footer section.
